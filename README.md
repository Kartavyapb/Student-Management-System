# Student-Management-System
A Djang based Student Management System with features for Student Registration, Attendance Tracking, Marks Management and Admin Control

The Student Management System is a Full Stack Django Web Application designed to manage complete Student Information in a Clean, Structured and Automated way. <br>
It provides a multi-step Student Registration Workflow, allowing Administrators to Add, Update, Delete and View Student Records with ease. <br>

This System is Useful for Schools, Colleges, Coaching Centers, Institutes or anyone wants a Structured Student Database with Automation Features like Auto Age Calculation, Auto Student ID generation and Fees Due Calculation.

## Key Features
### 1. Student Registration Process:
Student Information is added through separate, user-friendly forms:
* Student Personal Details
* Contact Information
* Parent Details
* Academic Information
* Course Details
* Fees Information
Each section updates the same Student records using sessions

### 2. Auto Generated Fields:
* Student ID (STU0001, STU0002, ....)
* Age (Calculated automatically when user enter the Date of Birth)
* Due Fees (Auto Calculated = Total Fees - Paid Fees)

### 3. Student CRUD Operations:
* Create -> Multi Step Registration Process
* Read -> Detailed Student Profile Page
* Update -> Crispy form-enabled update page
* Delete -> Delete Record Permanently
* List View -> Displays all Students with Search and Sorting

### 4. Interactive UI
* Fullly Style using Bootstrap
* Modern animated Carousel Homepage
* Clean Responsive list view with images
* Separate templates for each section

## ▶️ Usage Workflow
1. Open /home/
2. Click Add Student
3. Fill personal info → continue
4. Fill contact info → continue
5. Fill parents info → continue
6. Fill academic info → continue
7. Fill course info → continue
8. Fill fees info → save
9. View list of students
10. Open detail page → update/delete as needed
